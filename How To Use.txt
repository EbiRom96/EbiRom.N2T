📦 INSTALLATION
────────────────────────────────────────────────────────────────────
NuGet:    dotnet add package EbiRom.N2T
PM:       Install-Package EbiRom.N2T

🚀 QUICK START
────────────────────────────────────────────────────────────────────
using EbiRom.N2T;

// Convert integer numbers
string result = NumberToTextConverter.ConvertToText(
    1234567, 
    NumberToTextConverter.Language.Persian, 
    "ریال"
);

// Convert decimal numbers  
string decimalResult = NumberToTextConverter.ConvertToText(
    1234.56m,
    NumberToTextConverter.Language.Persian,
    "تومان"
);

// Convert without suffix
string simpleResult = NumberToTextConverter.ConvertToText(
    1500,
    NumberToTextConverter.Language.English
);

🌍 SUPPORTED LANGUAGES
────────────────────────────────────────────────────────────────────
• Persian   (فارسی)    - Language.Persian
• Arabic    (العربية)  - Language.Arabic  
• English   (English)  - Language.English
• Russian   (Русский)  - Language.Russian
• French    (Français) - Language.French
• German    (Deutsch)  - Language.German

📝 METHOD SIGNATURES
────────────────────────────────────────────────────────────────────
• string ConvertToText(long number, Language language, string suffix = "")
• string ConvertToText(decimal number, Language language, string suffix = "")

🔧 USAGE WITH SUFFIX
────────────────────────────────────────────────────────────────────
NumberToTextConverter.ConvertToText(1500, Language.Persian, "ریال")
→ "یک هزار و پانصد ریال"

NumberToTextConverter.ConvertToText(99.50, Language.English, "Dollars")  
→ "ninety-nine point fifty Dollars"

NumberToTextConverter.ConvertToText(1000000, Language.Arabic, "دينار")
→ "واحد مليون دينار"

⚠️ IMPORTANT NOTES
────────────────────────────────────────────────────────────────────
• Supports both positive and negative numbers
• Handles integers and decimals
• Optional suffix parameter for currency units
• Culture-aware formatting for each language
• High performance for large numbers

🆘 TROUBLESHOOTING
────────────────────────────────────────────────────────────────────
1. Make sure you have using EbiRom.N2T;
2. Check language enum spelling: Language.Persian (not Persian)
3. For decimals, use 'm' suffix: 123.45m
4. Suffix is optional - can be empty string ""
